# 🎲 Эрудит против ИИ

**Эрудит против ИИ** — это полноценная веб-реализация популярной настольной игры «Эрудит» (русский аналог Scrabble) с интеллектуальным противником, фоновой музыкой, анимированным интерфейсом и многопоточным поиском лучшего хода через Web Worker.

🔗 **Играть онлайн:** [kizrrum.github.io/erudit](https://kizrrum.github.io/erudit)  
📦 **Репозиторий:** [github.com/kizrrum/erudit](https://github.com/kizrrum/erudit)

---

## ✨ Особенности

*   🤖 **Умный ИИ** — противник анализирует доску и находит оптимальный ход с учётом бонусных клеток.
*   🧠 **Многопоточность** — поиск хода ИИ и подсказки вынесены в **Web Worker**, поэтому интерфейс не зависает во время расчётов.
*   🎵 **Фоновая музыка** — два режима на выбор: классические произведения (Бетховен, Моцарт) и атмосферный эмбиент.
*   🎨 **Визуальные эффекты** — анимация появления фишек, подсветка хода, эффект «встряски» при ошибке.
*   🔤 **Полный русский словарь** — все слова проверяются по загружаемому словарю (`words.txt`) с использованием префиксного дерева (Trie) для быстрого поиска.
*   🃏 **Джокер** — фишка «*» может заменить любую букву по выбору игрока.
*   📊 **Бонусные клетки** — точно как в оригинале:
    *   `x2 БУКВ` (DL) / `x3 БУКВ` (TL)
    *   `x2 СЛО` (DW) / `x3 СЛОВО` (TW)
*   ✅ **Соответствие оригинальным правилам** — точное количество и стоимость фишек, премия +15 очков за использование всех 7 фишек, обязательное прохождение первого слова через центр.

## 🎮 Как играть

### Цель
Набрать **200 очков** быстрее, чем ИИ.

### Управление

1.  **Выложить слово:**
    *   Нажмите на букву в своей руке (она подсветится).
    *   Кликните на свободную клетку на доске, чтобы поставить букву.
    *   Выкладывайте фишки, формируя слово по горизонтали или вертикали.
    *   Закончив, нажмите **«Готово»**.

2.  **Пропуск хода и замена фишек:**
    *   Нажмите **«Пропуск»**, чтобы открыть диалог замены.
    *   Укажите количество фишек (от 1 до 7), которые хотите заменить.
    *   Заменённые фишки возвращаются в мешок, а вы добираете новые, после чего ход переходит к ИИ.

3.  **Подсказка:**
    *   Нажмите **«💡»**, и ИИ найдёт лучший вариант хода для вас.

4.  **Музыка:**
    *   **🔊/🔇** — включить/выключить музыку.
    *   **🎼/🌿** — переключить режим (классика / эмбиент).

## 🛠️ Локальный запуск

Так как игра загружает словарь через `fetch`, для локальной работы требуется простой веб-сервер (недостаточно просто открыть `index.html` в браузере).

- Установленный [Node.js](https://nodejs.org/) (версия 14 или выше)
