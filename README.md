# 🎲 Эрудит против ИИ

Классическая игра «Эрудит» (Scrabble) против компьютера. Русский словарь, бонусные клетки, подсказки и музыка.

🔗 **Играть онлайн:** [kizrrum.github.io/erudit](https://kizrrum.github.io/erudit)

---

## 🎮 Как играть

- Нажмите на букву в руке → кликните на клетку доски → собирайте слово.
- Первое слово обязательно через центральную клетку ★.
- Каждая новая буква должна касаться уже стоящих.
- Бонусные клетки: `x2 БУКВ`, `x3 БУКВ`, `x2 СЛО`, `x3 СЛОВО`.
- Если использовали все 7 фишек за ход — +15 очков.
- Не можете ходить? Нажмите «Пропуск» — замените от 1 до 7 фишек.
- Кнопка 💡 — подсказка (ИИ покажет лучший ход).

Цель — набрать 200 очков раньше компьютера.

## 🛠️ Локальный запуск (если хотите играть без интернета)

Просто открыть `index.html` двойным щелчком **не получится** — браузер заблокирует загрузку словаря. Нужен веб-сервер.

### Способ 1: через Node.js (рекомендуется)

1. Убедитесь, что установлен [Node.js](https://nodejs.org/).
2. Откройте терминал (командную строку) в папке с игрой.
3. Выполните команду установки зависимостей:
   ```bash
   npm install
Запустите сервер:

   ```bash
node server.js
Откройте браузер и перейдите по адресу http://localhost:3000 — игра запустится.
